# CinemaLib

CinemaLib is a simple Android TV application which retrieves movie data from themoviedb.org.

It displays three movie categories: Popular, Top Rated, and Upcoming.

Clicking on each movie displays a preview with a poster and description.

The app allows you to change the language and theme.
